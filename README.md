# Solo AI Ops Kit — Free Edition 🚢
### Claude Code で「ひとりの仕事」を回す、日本語AIエージェント実務キット

> 読むだけのテンプレではありません。**Claude Code に入れて、すぐ動く**エージェントとスキルのセットです。
> このFree版は「まず動かして体験する」ための最小構成です。

---

## これは何？
ひとり会社・個人開発者が、Claude Code を「指示を出すだけで仕事が進む相棒」にするためのキットです。
**コマンドを覚える必要はありません。普通の日本語で話しかけるだけ。**

Free版に入っているもの:

| 種類 | 名前 | 役割 |
|------|------|------|
| エージェント | `orchestrator` | あなたの言葉を読み取り、適切な作業へ振り分ける統括役 |
| エージェント | `dev` | 設計・実装・レビューを小さく回す開発担当 |
| スキル | `research-brief` | テーマを調べて出典つきの要点に短時間でまとめる |

## 5分で動かす（クイックスタート）
1. このフォルダーの中の `.claude/` を、**あなたのプロジェクト直下にコピー**する
   ```
   your-project/
     └ .claude/
         ├ agents/    (orchestrator.md, dev.md)
         ├ skills/    (research-brief/SKILL.md)
         ├ hooks/     (block-external-actions.sh) ← 外向き操作を止めるガードレール
         └ settings.json                          ← 上のフックを有効化する設定
   ```
2. そのプロジェクトで Claude Code を開く
3. 普通に話しかける:
   - 「今日やることを整理して」
   - 「〇〇について調べてブリーフにして」→ `research-brief` が動く
   - 「このバグ直して」→ `dev` が動く

それだけです。エージェントは `.claude/agents/`、スキルは `.claude/skills/` に置くだけで認識されます。

## 設計思想（なぜ効くのか）
- **薄いオーケストレーター** — 統括役は自分で全部やらず、担当へ委譲する。だから破綻しにくい
- **外向きの行動は下書きで止める** — 公開・送信・デプロイは必ず一度あなたの確認を挟む
- **事実主義** — 調べ物は「たぶん」で埋めず、出典つきの確認できた事実だけを残す
